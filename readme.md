## Requirements
- Node
https://nodejs.org/

**Node.js 14.0.0 or newer is required.**

- Git
https://git-scm.com/

- Discord
https://discord.com/

- Discord Developer Portal
https://discord.com/developers/applications

## Installation

    clone or download

    cd <folder>

    npm outdated

    npm update

    node index.js
